HTML

  • The body element represents the visible content shown to the user.
  • The meta element contains data that is not visible to the client.
  • The doctype declaration lets the browser know what type of language to expect on the site
  • ul and li tags mean unordered list and list item respectively

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch: without the -b. will navigate you to the specified branch
  • SSH (secure shell network protocol) allows you to interface with certain remote services without having to constantly retype login information
  • ssh-agent is a program that keeps tracks of a user's identity keys
  • git revert _______: reverts to an earlier version. Useful when fixing bugs
  • Git Command Cheat Sheet
  • git add -A: stages all commits (changes) in the working branch
  • git commit: commits the changes. Use git commit -m "message" to add a short descriptive message describing the changes committed.
